# psql REPL

## connecting to a database
```bash
\c dbname username
```

## list all databases
```bash
\l
```

## list all tables in the current database
```bash
\dt
```

## describe a table
```bash
\d table_name
```

## list all schemas
```bash
\dn
```

## list all users and their roles
```bash
\du
```
